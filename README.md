# Telefonkatalogen

Dette prosjektet er en telefonkatalog-applikasjon laget med Python. Den lar brukere legge til, søke etter og fjerne kontakter fra en database.

## Funksjoner
- Legg til nye kontakter
- Søk etter eksisterende kontakter
- Slett kontakter
